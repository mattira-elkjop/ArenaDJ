Quick Start Guide · Version 7
What is Arena DJ?

Arena DJ is a browser-based music controller for ice hockey (and other sports) arenas. Assign YouTube music to specific game events — goals, power plays, intermissions and more — then trigger them with a single button tap or keyboard shortcut. Songs shuffle automatically with configurable fade-in and fade-out.

YouTube Ads & Best Playback Setup

Arena DJ streams audio via the hidden YouTube player. YouTube may play ads before songs if you are not signed in or do not have YouTube Premium.

Ads cannot be skipped or suppressed programmatically. For uninterrupted arena use:
  • Best: Sign in to YouTube in this browser with an active YouTube Premium subscription.
  • Good: Use a browser with an ad blocker (uBlock Origin, AdGuard, etc.).
  • Note: The YouTube player is hidden (audio only) — visual ads won't show, but audio ads will play over the PA if they occur.

Sign in to YouTube at any time via the notice banner at the top of the app.

Getting Started
1 — Add Songs (Settings → Song Library → + ADD SONG)

Click + ADD SONG to open the song browser. You can either search YouTube directly (requires a free API key — see Settings → YouTube API Key) or paste any YouTube URL. Once a video loads, use the clip scrubber to set exactly where the song starts and ends, then assign it to one or more events and click + ADD TO LIBRARY.

2 — Set Defaults (Settings → Global Defaults)

Set how long songs play before auto-advancing (Default Play Duration — used for any song without an explicit clip end set), fade-out and fade-in durations, and the Announcement Fade duration. Per-song clip times override the global default.

3 — Go Live

Switch to the Live tab. Press any event button to start playing. The Now Playing bar at the top is always visible across all pages.

Adding Songs — Clip Scrubber

When previewing a song in the Add Song modal, the clip scrubber lets you define exactly what part of the video will play in the arena:

  • Cyan handle (left) — start position. Drag to set, or play to the right moment and click ⊙ Mark Start.
  • Amber handle (right) — end position. Same — drag or use ⊙ Mark End.
  • Loop from Start — when ON, playback loops back to your start marker when the video ends, so you can keep listening while finding the right end point.
  • Click anywhere on the scrubber track to seek the player to that position.

If you leave the end handle at the very end of the video, the song will use the Default Play Duration from Global Defaults for auto-advance timing.

Live Page
Event Button Zones

Each button has three clickable zones:

  • Left edge (↺) — Replay the current song from its start time.
  • Middle — Play / Pause / Resume toggle for that event.
  • Right edge (⏭) — Skip to the next song immediately.
Up Next

The bottom strip shows the next queued song. Click ⟳ to re-roll to a different random song without affecting what's currently playing.

Keyboard Shortcuts

All shortcuts can be customised in Settings → Keyboard Shortcuts. SPACE is always reserved for play/pause. The three Global Actions (Next Song, Replay, Announcement Fade) are also configurable and support modifier combinations like Ctrl+→.

Announcement Fade

Press the Announcement Fade shortcut (default: F) to slowly fade the music out over a configurable number of seconds (default: 8s), then pause. This gives a smooth, professional transition before a PA announcement. Press SPACE afterwards to resume from where it stopped.

The fade duration can be adjusted in Settings → Global Defaults → Announcement Fade.

Now Playing Bar

Always visible at the top of every page. Shows the current song, event, and a full-width scrubber:

  • Click or drag the track to seek.
  • The highlighted zone shows the configured clip window (start → end).
  • The amber handle on the right edge of the window marks the end time — drag it to temporarily extend or shorten the current song without changing saved settings.
  • Time labels show: elapsed · clip window range · total video length.
Song Library
Searching & Filtering

Use the search box to filter by title in real time. Use Filter by Event to show only songs for specific events. Click ⇅ or the Title column header to toggle sort order.

Editing

Click ✎ EDIT on any row to edit inline — title, URL/video ID, event, start time, and duration — without deleting and re-adding.

One Song, Multiple Events

When adding a song you can select multiple events. A separate entry is created per event, so each can have its own clip times. Edit them individually afterwards if needed.

YouTube API Key

The in-app YouTube search requires a free API key from Google. Without one, you can still add songs by pasting a YouTube URL directly. To set up search, go to Settings → YouTube API Key and follow the step-by-step instructions there.

Import & Export

Use Export .JSON to save a full backup of your songs, settings, and shortcuts. Use Import .JSON to restore on the same or a different device. Files from older versions are migrated automatically.

Tip: Export a backup before making large changes to your library.

Event Reference
  • Match Intro — Players entering the ice, up to puck drop.
  • Warm-Ups — Pre-game and between-period music. Always plays the full song — start time and duration settings do not apply.
  • Intermission — Mid-game stoppages. Uses clip start and duration settings.
  • Game Over — Final horn, end of game.
  • Goal — Home — Home team scores.
  • Goal — Away — Away team scores.
  • Power Play — Home team on power play (opponent penalised).
  • Box Play — Home team short-handed (own player penalised).
  • Timeout — Either team calls timeout.
  • Hard Hit — Big body check moment.
  • Maintenance — Play stopped for ice or board repair.

🔑 YOUTUBE API KEY

Required for in-app YouTube search. Free — no billing needed for typical use. Your key is stored locally and never sent anywhere except YouTube's own API.

HOW TO GET YOUR KEY
  1. Go to Google Cloud Console → YouTube Data API v3
  2. Sign in with any Google account, then click Enable
  3. In the left sidebar go to APIs & Services → Credentials
  4. Click + Create Credentials → API key
  5. Copy the key (starts with AIza…) and paste it below
  6. Optional but recommended: click Edit API key → restrict to "YouTube Data API v3" only
