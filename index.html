ARENA DJ
Quick Start Guide · Version 6
What is Arena DJ?

Arena DJ is a browser-based music controller designed for ice hockey (and other sports) arena environments. It lets you assign music playlists to specific game events — goals, power plays, intermissions and more — and trigger them with a single button click or keyboard shortcut, with automatic fade-in, fade-out, and shuffle between songs.

Tip: Always launch using START_Arena_DJ.bat (Windows). This starts a local web server required for YouTube playback. Opening the HTML file directly will not work.
YouTube Ads & Best Playback Setup

Arena DJ streams audio via the YouTube player. YouTube may play ads before songs start if you are not signed in to a Google account, or do not have YouTube Premium.

There is no way to skip or suppress YouTube ads programmatically. For uninterrupted arena use, choose one of the following:
  • Best: Sign in to YouTube in this browser and have an active YouTube Premium subscription — no ads anywhere, including embeds.
  • Good: Use a browser with an ad blocker extension installed (uBlock Origin, AdGuard, etc.).
  • Note: The YouTube player in Arena DJ is hidden (audio only), so visual ads won't clutter your screen — but audio ads will still be audible over the PA system if they play.

You can sign in to YouTube at any time by clicking Sign in to YouTube ↗ in the notice banner at the top of the app, then reload the page.

Getting Started
1 — Add Songs (Settings → Song Library)

Each song needs a YouTube URL, a title, at least one event assignment, and optionally a Start At (skip intro) and Play Duration (how long to play before auto-advancing). You can assign a single song to multiple events at once — it creates a separate entry per event, so you can later set different start times and durations for each.

2 — Set Defaults (Settings → Global Defaults)

Set how long songs play before advancing (Default Play Duration), how long the fade-out lasts, and how long the fade-in takes when a song starts. Per-song durations override the global default.

3 — Go Live

Switch to the Live tab. Press any event button to start playing a shuffled song from that event's pool. The Now Playing bar at the top shows what's playing across all pages.

Live Page
Event Button Zones

Each button has three clickable zones:

  • Left edge (↺) — Replays the current song from its configured start time.
  • Middle — Play / Pause / Resume toggle for that event.
  • Right edge (⏭) — Skips to the next song in that event's queue immediately.
Up Next

The bottom strip of every button shows the next song queued for that event. Click ⟳ to re-roll it to a different random song without affecting playback.

Keyboard Shortcuts

Shortcuts can be customised in Settings. SPACE is always global play/pause for the last active event.

Now Playing Bar

The bar at the top is always visible, even on the Settings page. It shows the current song, event, and a scrubber you can interact with:

  • Click or drag anywhere on the track to seek to that position in the video.
  • The lighter shaded area represents the configured active window (start → end) within the full video.
  • The amber handle (⬜) on the right edge of the window marks the configured end time. Drag it left or right to temporarily shorten or extend how long the current song plays — this does not change your saved settings.
  • The time labels show: elapsed in window · window range in video · total video length.
Song Library
Search

Type in the search box to filter songs by title in real time.

Filter by Event

Click Filter by Event to open a dropdown and select one or more events. Only songs assigned to those events will be shown. A badge shows how many filters are active.

Sort

Click the ⇅ sort button or the Title column header to toggle between alphabetical (A→Z / Z→A) and date-added order.

Editing a Song

Click ✎ EDIT on any row to edit it inline. You can change the title, URL/video ID, event, start time, and play duration without deleting and re-adding.

One Song, Multiple Events

When adding a song you can tick multiple events. A separate entry is created per event, so each can have its own start time and duration. Edit them individually afterwards if needed.

Import & Export

Use Export .JSON to save a complete backup of all your songs, settings, and keyboard shortcuts to a file. Use Import .JSON to restore it on the same or a different computer.

Files from older versions of Arena DJ are automatically migrated when imported — you will see a notification if migration occurred.

Tip: Export a backup before making large changes to your song library.
Event Reference
  • Match Intro — Players entering the ice, up to puck drop.
  • Warm-Ups — Pre-game warmup and between-period music. Always plays the full song from beginning to end — no start time or duration settings apply.
  • Goal — Home — Home team scores.
  • Goal — Away — Away team scores.
  • Power Play — Home team on power play (opponent penalised).
  • Box Play — Home team short-handed (own player penalised).
  • Timeout — Either team calls timeout.
  • Hard Hit — Big body check moment.
  • Intermission — Mid-game stoppages (broken glass, ice maintenance, etc.). Uses configured start times and durations.
  • Maintenance — Play stopped for ice or board repair.
  • Game Over — Final horn, end of game.
